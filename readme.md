# Api BlackTag - Não oficial

Uma api para comunicação entre a blacktag e sites tercerizados<br/>

## Principais funcionalidades
- Verificar quantidade de ingressos em eventos 🔜
- Envio de ingressos 🔜
- Compra de ingressos 🔜
